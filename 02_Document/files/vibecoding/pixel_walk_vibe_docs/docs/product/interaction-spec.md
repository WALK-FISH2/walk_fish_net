# 滚动交互规格 Interaction Spec

## 1. 全局进度

```text
globalProgress ∈ [0, 1]
```

```text
0.00–0.30 陆地
0.30–0.38 下潜
0.38–0.66 深海
0.66–0.80 海洋到星空
0.80–1.00 星空
```

## 2. 确定性

同一滚动位置应得到基本一致的场景颜色、元素位置、旋转角度、粒子转换状态、DOM 阶段和导航状态。环境循环可以随时间变化，但不得改变叙事状态。

## 3. 正向和反向

```text
向下：陆地 → 深海 → 星空
向上：星空 → 深海 → 陆地
```

不得在回滚时瞬移、重新生成全部粒子、闪白或让导航错位。

## 4. 视差

```text
远景 0.10–0.20
中景 0.30–0.50
近景 0.60–0.80
前景 0.90–1.20
```

## 5. 陆地下潜

| 局部进度 | 事件 |
|---:|---|
| 0.00 | 陆地稳定 |
| 0.15 | 天空略暗 |
| 0.30 | 底部出现水面 |
| 0.50 | 水线经过内容 |
| 0.70 | 屏幕大部分入水 |
| 0.85 | 气泡出现 |
| 1.00 | 深海稳定 |

## 6. 海洋到星空

| 全局进度 | 事件 |
|---:|---|
| 0.66–0.69 | 水流增强 |
| 0.69–0.72 | 镜头上升 |
| 0.72–0.75 | 三层浪花 |
| 0.75–0.78 | 背景翻转 |
| 0.78–0.80 | 泡沫转星点 |

同一粒子对象建议具有 underwaterState、foamState 和 starState，通过插值转换，不销毁后重建。

## 7. DOM 稳定

使用独立容器：

```text
sceneWorld
contentOverlay
fixedUI
```

仅 `sceneWorld` 翻转。导航、进度、正文和关键按钮不参与背景旋转。

## 8. 输入

支持鼠标滚轮、触控、触控板、键盘 PageDown/PageUp 和进度节点点击。

禁止禁用默认滚动、长时间阻塞主线程、强制整页吸附和无限惯性滞后。

## 9. Reduced Motion

```text
陆地静态场景
→ 像素遮罩
→ 深海静态场景
→ 像素遮罩
→ 星空静态场景
```

取消大幅旋转、快速视差、镜头摇晃和大规模粒子涌动。
