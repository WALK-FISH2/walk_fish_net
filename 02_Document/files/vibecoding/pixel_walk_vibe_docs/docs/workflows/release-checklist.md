# Release Checklist

## 内容

- [ ] 姓名和联系方式正确；
- [ ] 文章真实；
- [ ] 程序均为本人编写或贡献明确；
- [ ] 无虚假在线功能和占位文本；
- [ ] 无无效链接。

## 安全与素材

- [ ] 无 Token、密码、私有接口和内部截图；
- [ ] iframe 最小权限；
- [ ] 外部链接 noopener；
- [ ] 素材原创或许可清晰；
- [ ] 无游戏和电影原始素材；
- [ ] 字体和音效许可存在。

## 质量

- [ ] typecheck；
- [ ] lint；
- [ ] test；
- [ ] e2e；
- [ ] build；
- [ ] 静态预览；
- [ ] 路由刷新；
- [ ] 404；
- [ ] Reduced Motion；
- [ ] 375px；
- [ ] 键盘；
- [ ] 控制台无错误。

## SEO 和部署

- [ ] title、description、canonical、Open Graph；
- [ ] sitemap、robots、favicon；
- [ ] 输出目录和构建平台配置正确；
- [ ] 自定义域名、HTTPS 和缓存；
- [ ] 发布后冒烟测试。

## 文档

- [ ] README；
- [ ] CHANGELOG；
- [ ] tasks；
- [ ] ADR；
- [ ] 发布版本号。
