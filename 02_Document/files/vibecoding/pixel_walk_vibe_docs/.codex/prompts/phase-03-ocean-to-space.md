# 提示词：第三轮海洋到星空

本轮只处理全局进度 0.66–0.80，不修改文章系统、程序系统、详情页和其他已完成模块。

```text
0.66–0.69 水流增强
0.69–0.72 镜头上升
0.72–0.75 三层浪花
0.75–0.78 背景翻转
0.78–0.80 泡沫转星点
```

要求：

1. 至少前、中、远三层浪，波形不同；
2. 使用泡沫对象池和固定随机种子；
3. 同一批泡沫对象转成星点；
4. 气泡转星尘，高光转银河；
5. 只翻转 sceneWorld，导航和 DOM 保持水平；
6. 桌面旋转 150–180 度，移动端 90–120 度；
7. Reduced Motion 不旋转；
8. 用户停止滚动时停止；
9. 向上滚动完整倒放；
10. 不劫持滚轮；
11. 构建和静态输出继续成功。

必须提供时间线配置、容器层级、对象复用方案、Reduced Motion 方案、性能结果和 72%/76%/80% 截图。
