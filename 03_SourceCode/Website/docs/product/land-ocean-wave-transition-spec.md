# 陆地到海洋：翻涌海浪过渡规格

状态：已实施并通过正式验收
建议里程碑：M4.5
前置条件：M4 下潜与深海已经完成
目标：在不推翻现有 M4 的前提下，让陆地到海洋的交接更丝滑

---

## 1. 目标效果

当前陆地到海洋已经具备水位上涨与下潜结构。本轮增加一段短而明确的翻涌海浪：

```text
陆地环境变冷
→ 远处水面出现
→ 海浪逐渐接近
→ 前、中、远三层浪峰翻涌
→ 泡沫与水雾覆盖场景接缝
→ 镜头穿过浪峰下方
→ 水面退到顶部
→ 进入深海
```

本段不进行世界翻转，不生成星点，也不负责银河或星空。

---

## 2. 进度区间

保持现有 M4 总区间不变，建议重新分配内部节奏：

```text
0.300–0.315  陆地环境变冷
0.315–0.328  远处水面和第一层小浪出现
0.328–0.342  浪潮接近，水位继续上涨
0.342–0.355  三层浪峰翻涌并经过屏幕中部
0.355–0.368  泡沫、水雾覆盖接缝，镜头穿浪
0.368–0.380  完全入水，水面退到屏幕顶部
```

若当前项目已使用其他内部区间，可以微调，但不得改变 M4 与 M5 的全局边界。所有参数必须集中放入现有 story/scene 配置。

---

## 3. 三层海浪

### 3.1 远浪

- 最先出现；
- 振幅较小；
- 颜色较暗；
- 速度较慢；
- 位于场景后方；
- 用于提示水域正在接近。

### 3.2 中浪

- 形成主要水面轮廓；
- 振幅中等；
- 具有清晰波峰；
- 带少量浅色反光；
- 是视觉上的主水面。

### 3.3 前浪

- 最后进入；
- 移动更快；
- 具有明显泡沫；
- 短暂遮挡画面中下部；
- 负责遮盖陆地与水下场景的接缝。

三层浪不能复制同一条正弦曲线。至少要在以下参数上区分：

```text
振幅
周期
相位
水平速度
透明度
像素块尺寸
泡沫密度
```

---

## 4. 波浪形态

目标是“像素浪峰”，不是光滑矢量线。

要求：

- 波峰边缘呈阶梯状像素形态；
- 波谷具有不规则变化；
- 局部可出现卷曲浪头；
- 不要求真实流体模拟；
- 不能只有一条水平波纹；
- 不使用全屏视频；
- 不直接使用第三方海浪素材。

可采用：

- 程序化折线；
- 多段曲线像素化；
- PixiJS Graphics；
- 原创像素纹理平铺；
- 固定种子的轻微噪声。

---

## 5. 泡沫与水雾

### 泡沫

主要出现在：

- 波峰边缘；
- 前浪破碎位置；
- 镜头穿浪时；
- 屏幕中部的过渡遮挡区域。

要求：

- 由小型像素簇组成；
- 大小不完全一致；
- 不全部为圆形；
- 使用固定随机种子；
- 能够反向倒放；
- 使用对象池或复用机制；
- 不在每帧大量创建对象。

### 水雾

只作为辅助：

- 透明度低；
- 生命周期短；
- 不模糊正文；
- 不持续覆盖导航；
- 移动端减少或取消。

---

## 6. 遮挡场景接缝

当泡沫覆盖画面中部时：

1. 陆地背景降低对比度；
2. 水下色调逐渐接管；
3. 陆地前景变成剪影；
4. 水下光束和气泡开始出现；
5. 泡沫退去后，画面已经处于水下。

不得出现：

- 白屏；
- 瞬间换背景；
- 陆地元素突然消失；
- 水下元素突然全部出现；
- 导航被旋转；
- DOM 内容长时间不可读。

---

## 7. 镜头穿浪

建议组合：

- 前浪从下方向上推进；
- 场景世界轻微纵向位移；
- 折射仅出现在水线附近；
- 泡沫覆盖中间接缝；
- 水下颜色由下向上扩展；
- 入水后水面继续退到顶部。

禁止：

- 180 度旋转；
- 持续镜头抖动；
- 强制滚轮吸附；
- 自动播放完整动画。

---

## 8. DOM 与固定 UI

以下内容始终保持水平：

- 顶部导航；
- 右侧阶段进度；
- 文章和 Programs 的正文；
- 可点击按钮；
- 无障碍提示。

浪花可以短暂经过 DOM，但不得改变布局、阻止点击或长时间降低文字对比。粒子层需要使用合适的 `pointer-events`。

---

## 9. 可逆性

向上滚动时必须完整倒放：

```text
深海
→ 水面从顶部靠近
→ 泡沫重新汇聚
→ 前浪、中浪、远浪退回
→ 陆地恢复颜色
```

不得：

- 重建随机浪花；
- 泡沫突然消失；
- 波浪相位跳变；
- 陆地在最后一帧突然出现；
- 反向动画缺少某一层浪。

---

## 10. 移动端

移动端必须保留：

- 远浪；
- 主浪；
- 少量泡沫；
- 镜头穿浪；
- 水面退到顶部。

可以简化：

- 前浪细节；
- 泡沫密度；
- 水雾；
- 局部折射；
- 视差强度。

不得简化成直接切换背景颜色。

---

## 11. Reduced Motion

Reduced Motion 下：

```text
陆地
→ 一层静态像素浪幕向上遮挡
→ 水下静态场景
```

取消：

- 多层高速浪花；
- 水雾；
- 大幅视差；
- 动态折射；
- 镜头晃动。

仍需保留明确的陆海交接感。

---

## 12. 性能

- 波浪几何复用；
- 泡沫使用对象池；
- 不每帧创建大量对象；
- DPR 上限保持 2；
- 页面隐藏时暂停环境循环；
- 移动端减少泡沫；
- 不增加全屏视频；
- 不引入新的大型动画框架。
