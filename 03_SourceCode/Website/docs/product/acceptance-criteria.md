# 验收标准 Acceptance Criteria

## 静态站点

- [ ] 不运行 Node 服务也能访问生产输出；
- [ ] 首页、文章、程序和关于我可访问；
- [ ] 每篇文章和程序可直接刷新；
- [ ] 404 正常；
- [ ] 资源路径在目标环境正常。

## 做点啥呢

- [ ] 所有条目均为本人编写或贡献明确；
- [ ] 每项有状态、演示类型和限制；
- [ ] 静态演示不会伪装完整后端；
- [ ] 源码缺失时不显示空按钮；
- [ ] 演示失败不影响详情阅读；
- [ ] iframe 使用最小权限。

## 首页场景

- [ ] 0% 明显是陆地；
- [ ] 陆地 0%–30% 的中间进度中，引导人物的大部分行走路径不被 Hero、三张文章路牌或路牌柱遮挡；
- [ ] `≤980px` 的三张文章路牌可横向滚动和键盘聚焦，且不制造整页横向滚动；
- [ ] 50% 明显是深海；
- [ ] Programs 标题、三个档案终端和总入口按顺序可读，不发生标题/卡片或卡片/卡片持续重叠；
- [ ] 90% 明显是星空；
- [ ] 不看标题也能区分世界；
- [ ] 陆地到深海不是直接切色；
- [ ] M4.5 的陆海交接有三层浪花与泡沫遮挡，M5 不重复绘制海浪；
- [ ] 海洋到星空依靠颜色、光束与粒子形态平滑过渡；
- [ ] 海空过渡由同一固定 `MorphParticle` 池依次表现气泡、星点和少量流星，不清空后重建无关星点；
- [ ] 64%、66%、69%、72%、75.5%、78.5%、80% 和 82% 的形态变化连续，82%→64% 可恢复深海；
- [ ] 桌面/375px/Reduced Motion 的流星比例分别不超过 `5.5%`/`2.2%`/`0`；
- [ ] Canvas 背景世界和 DOM 在全部进度始终保持水平，不发生旋转；
- [ ] 向上滚动可完整倒放；
- [ ] 停止滚动时状态稳定。

## 移动端与无障碍

- [ ] 375px 无横向滚动；
- [ ] 按钮可触控且不依赖 hover；
- [ ] 粒子按视口和动效偏好降级；
- [ ] Skip link、可见焦点、键盘导航、图片 alt；
- [ ] Canvas aria-hidden；
- [ ] Reduced Motion；
- [ ] `?motion=full` 在系统 Reduced Motion 环境中恢复标准阶段高度，同时保持内容层无重叠；
- [ ] 不自动播放声音。

## 质量

- [ ] 类型、Lint、单元、集成和 E2E 通过；
- [ ] 生产构建通过；
- [ ] 控制台无未处理错误；
- [ ] 页面切换后无重复时间线；
- [ ] 无明显内存持续增长；
- [ ] 文档同步完成。
