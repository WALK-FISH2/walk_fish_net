# 验收标准 Acceptance Criteria

## 静态站点

- [ ] 不运行 Node 服务也能访问生产输出；
- [ ] 首页、文章、程序和关于我可访问；
- [ ] 每篇文章和程序可直接刷新；
- [ ] 404 正常；
- [ ] 资源路径在目标环境正常。

## 做点啥呢

- [ ] 所有条目均为本人编写或贡献明确；
- [ ] 每项有状态、演示类型和限制；
- [ ] 静态演示不会伪装完整后端；
- [ ] 源码缺失时不显示空按钮；
- [ ] 演示失败不影响详情阅读；
- [ ] iframe 使用最小权限。

## 首页场景

- [ ] 0% 明显是陆地；
- [ ] 陆地 0%–30% 的中间进度中，引导人物的大部分行走路径不被 Hero、三张文章路牌或路牌柱遮挡；
- [ ] `≤980px` 的三张文章路牌可横向滚动和键盘聚焦，且不制造整页横向滚动；
- [ ] 50% 明显是深海；
- [ ] Programs 标题、三个档案终端和总入口按顺序可读，不发生标题/卡片或卡片/卡片持续重叠；
- [ ] 90% 明显是星空；
- [ ] 不看标题也能区分世界；
- [ ] 陆地到深海不是直接切色；
- [ ] M4.5 的陆海交接有三层浪花与泡沫遮挡，M5 不重复绘制海浪；
- [ ] 海洋到星空依靠颜色、光束与粒子形态平滑过渡；
- [ ] 海空过渡由同一固定 `MorphParticle` 池依次表现气泡、星点和少量流星，不清空后重建无关星点；
- [ ] 64%、66%、69%、72%、75.5%、78.5%、80% 和 82% 的形态变化连续，82%→64% 可恢复深海；
- [ ] 桌面/375px/Reduced Motion 的流星比例分别不超过 `5.5%`/`2.2%`/`0`；
- [ ] M5.5 固定流星全部朝右下，拖尾位于头部左上，原数量、位置和视差关系不变；
- [ ] M5.5 高速流星不读取滚动进度，只在星空中偶发运行，离开章节、切后台和卸载时完整清理；
- [ ] M5.5 行星具有核心、内亮、外扩散、环境散射和前后分层行星环；
- [ ] M5.5 三层浪使用透明生产 sprite，前/中/后轮廓、相位、速度和透明度有明显差异；横向泡沫带与水下气泡簇用途不混淆且无明显接缝；
- [ ] Canvas 背景世界和 DOM 在全部进度始终保持水平，不发生旋转；
- [ ] 向上滚动可完整倒放；
- [ ] 停止滚动时状态稳定。

## 移动端与无障碍

- [ ] 375px 无横向滚动；
- [ ] 按钮可触控且不依赖 hover；
- [ ] 粒子按视口和动效偏好降级；
- [ ] Skip link、可见焦点、键盘导航、图片 alt；
- [ ] Canvas aria-hidden；
- [x] 普通网址在无 URL 覆盖、无已保存偏好时直接进入完整动画，不要求添加 `?motion=full`；
- [x] fixed UI 提供可触控、可键盘聚焦、焦点可见且标签明确的完整/简化动画开关；
- [x] 用户主动选择在刷新、站内跳转和再次访问后保持；本地存储不可用时页面仍正常且控制台无未处理错误；
- [x] 模式优先级为有效 URL 参数 > 已保存选择 > 完整动画默认值，`?motion=full` 和 `?motion=reduce` 均不污染已保存偏好；
- [x] 系统 `prefers-reduced-motion: reduce` 可被识别并在完整模式下显示简化建议，一次操作即可切换；
- [x] 已保存简化模式在场景初始化前生效，不短暂播放完整视差、浪花或动态流星；
- [x] 简化动画保留三个可辨识世界与完整内容，关闭高速视差和动态流星并完成生命周期清理；
- [x] 运行中切换模式保持当前进度、章节高度、内容顺序和世界 0°，不产生重复时间线、RAF、timer 或监听器；
- [x] `?motion=full` 在任意系统偏好下恢复标准阶段高度，`?motion=reduce` 进入同一简化分支，两者均保持内容层无重叠；
- [ ] 不自动播放声音。

## 质量

- [ ] 类型、Lint、单元、集成和 E2E 通过；
- [ ] 生产构建通过；
- [ ] 控制台无未处理错误；
- [ ] 页面切换后无重复时间线；
- [ ] 无明显内存持续增长；
- [ ] 文档同步完成。
