# Review Checklist

## 需求

- [ ] 修改与当前任务一致；
- [ ] 没有未经要求的功能；
- [ ] 没有破坏“做点啥呢”的本人程序语义；
- [ ] 非目标没有被偷偷引入。

## 架构

- [ ] 静态部署仍可用；
- [ ] 内容与 Canvas 分离；
- [ ] 滚动进度来源唯一；
- [ ] 演示已隔离；
- [ ] 销毁逻辑存在。

## 代码

- [ ] 类型清晰；
- [ ] 没有无理由 any；
- [ ] 没有无关大重构；
- [ ] 没有硬编码秘密；
- [ ] 错误有降级。

## 视觉和动画

- [ ] 陆地、深海、星空明显不同；
- [ ] 不依赖纯色和普通矩形；
- [ ] 像素边缘清晰；
- [ ] 导航可读；
- [ ] 移动端无溢出；
- [ ] 动画可逆；
- [ ] 不劫持输入；
- [ ] Reduced Motion 有效；
- [ ] 回滚不重置粒子。

## 测试和文档

- [ ] 类型检查；
- [ ] Lint；
- [ ] 测试；
- [ ] 构建；
- [ ] 静态预览；
- [ ] 详情页刷新；
- [ ] 控制台无错误；
- [ ] tasks、ADR、README 和 CHANGELOG 已更新。
