# M5.5-A 星空视觉抛光任务单

> 建议项目路径：`docs/workflows/m5-5-starry-sky-polish-task.md`
>
> 前置条件：M5 已完成。本任务只做增量修改，不重新搭建星空场景。

---

## 任务目标

在不改变 M5 滚动叙事和章节结构的前提下，完成固定流星方向修正、随机高速流星和星体光环重做。

## 任务范围

### T1 固定流星方向

- 定位现有固定流星实现。
- 保留元素数量、分布和原视差关系。
- 调整为左上到右下，拖尾位于头部左上侧。

### T2 随机高速流星

- 新增独立 overlay，不挂入被滚动 transform 的背景层。
- 只在星空 section 可见时运行。
- 任意时刻最多 1~2 颗。
- 使用 1~3 秒随机间隔、快速运动、短尾迹、快速淡出；该间隔由项目所有者实际试用后确认。
- 页面隐藏、离开 section 或组件卸载时暂停并清理。
- 支持 reduced-motion 降级。

### T3 光环层次

- 将主要星体/右上行星拆为主体、内光、外光和环形层。
- 增强层间尺寸、明度与透明度差。
- 行星环处理前后遮挡关系。
- 避免大范围 blur 和两个近似圆层简单叠加。

## 推荐技术

### 流星

优先：Canvas + `requestAnimationFrame`。

建议对象字段：

- `x`, `y`
- `vx`, `vy`
- `life`, `maxLife`
- `trailLength`
- `headSize`
- `opacity`

建议配套：

- `IntersectionObserver` 控制 section 激活；
- `visibilitychange` 控制后台暂停；
- 单一 RAF 循环；
- timer、observer、RAF 完整 cleanup。

### 光环

根据现有实现选择 DOM、SVG 或 Canvas。无论技术方案如何，都要形成可辨认的多层结构，而不是复制贴图。

## 不允许

- 改动 M5 文案、导航和章节顺序；
- 将随机流星绑定到 scroll progress；
- 流星在非星空章节出现；
- 每次进入星空区新增一个未清理的 RAF；
- 为实现光环而使用大面积模糊滤镜破坏像素风。

## 验收

- [x] 固定流星方向正确。
- [x] 随机流星少量、偶发、快速、朝右下。
- [x] 滚动时随机流星不随背景整体移动。
- [x] 离开星空区后停止运行。
- [x] 多次进入章节后频率不叠加。
- [x] 光环层次清楚，行星环前后关系正确。
- [x] M5 原有交互和视觉结构无回归。
