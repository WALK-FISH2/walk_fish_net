# M5.5-B 三层浪花与气泡抛光任务单

> 建议项目路径：`docs/workflows/m5-5-wave-bubble-polish-task.md`
>
> 说明：虽然修改区域位于陆地到海洋过渡，但实施时间在 M5 完成之后，因此归入 M5.5，不再使用 M4.5 命名。

---

## 任务目标

保留 M5 基线中的三层浪花结构与滚动节奏，使用新参考素材改善浪花轮廓、泡沫细节和气泡分布。

## 参考素材

目录：`assets/reference/m5-5/`

- `wave-large-01.png`
- `wave-medium-02.png`
- `wave-small-03.png`
- `bubble-foam-band-01.png`
- `bubble-clusters-02.png`
- `bubble-foam-band-03.png`

这些文件是参考源，不应未经处理直接作为最终生产素材。

## 方案

使用“处理后的透明 sprite + 轻量代码运动”。

代码负责：

- 三层定位、裁切和 z-index；
- 轻微横向漂移；
- 不同相位的小幅上下起伏；
- 响应式适配；
- 平铺接缝处理；
- reduced-motion 降级。

## 三层要求

### L1 前景

- 最大、最清晰、卷浪最明显；
- 泡沫细节最多；
- 承担主要遮挡和自然衔接。

### L2 中景

- 高度和细节低于前景；
- 速度、相位与前景不同；
- 作为连接层，不抢主视觉。

### L3 背景

- 最低、最轻、最简洁；
- 以平缓海面和稀疏碎泡为主；
- 透明度最低。

## 气泡要求

- 横向气泡带用于浪尖和海面边界泡沫。
- 竖向气泡簇用于海水内部上升气泡。
- 不得将竖向气泡簇均匀铺满海面。
- 气泡密度应有局部变化，避免规则复制。

## 素材处理要求

- 去黑底并输出透明背景。
- 裁掉多余空白。
- 保留大像素块，不做平滑插值。
- 必要时统一色板，使其匹配现有海洋蓝色体系。
- 横向素材需要无明显接缝；建议制作多个变体或错位拼接。
- 生产素材与参考素材分目录保存，避免运行时误用原始参考图。

## 集成约束

- 不改变 M5 原有滚动区间、section 高度和进度语义。
- 不因贴图尺寸导致布局抖动或章节跳跃。
- 浪花只在既有过渡区域内显示。
- 不遮挡按钮、导航或主体文字。

## 验收

- [x] 三层轮廓和运动差异明显。
- [x] 新浪花比 M5 基线更自然、美观。
- [x] 气泡与泡沫使用位置合理。
- [x] 无明显横向接缝和机械重复。
- [x] 超宽屏与移动端无空白露出。
- [x] M5 滚动节奏和章节位置无回归。
